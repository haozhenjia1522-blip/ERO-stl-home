## Run the Web App (Vite)

### Prerequisites
- Install **Node.js** (LTS recommended). npm comes with Node.js.

### Start (Development)
Open a terminal at the **repo root** and run:

```bash
cd web
npm install
npm run dev
````

After `npm run dev`, the terminal will print a local URL (for example `http://localhost:5173/`).
**Click the URL** (or copy-paste it into your browser) to open the web app.

### Build (Production)

```bash
cd web
npm run build
```

### Preview the Production Build

```bash
cd web
npm run preview
```
